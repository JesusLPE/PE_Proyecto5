# BoleteriaExpress
- Boleteria de Bus

Funcionalidades principales ya implementadas:

- Pantalla de inicio con opciones: Ingresar, Registrarse, Panel Admin y Salir.
- Registro y login de usuarios (datos guardados solo en memoria).
- Usuario admin especial (admin/0000) y usuarios normales.
- Panel de usuario: buscador básico de rutas (aún no funcional), opciones rápidas y compra de asientos.
- Panel de administrador: ver usuarios registrados, vender boletos y otras opciones pendientes.
- Selección gráfica de asientos: mapa de bus con 52 asientos (10 filas × 4 columnas), puedes "comprar" asientos que se marcan como ocupados (en memoria).

Lo que aún está pendiente ("próximamente"):

- Búsqueda real de rutas, cooperativas y horarios.
- Historial de compras y registro de ventas.
- Persistencia de datos (usuarios y boletos vendidos se pierden al cerrar la app).
